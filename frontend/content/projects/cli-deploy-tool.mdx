## 项目背景

传统 CMS（如 WordPress、Strapi）对于以 Markdown 为主的技术博客来说过于复杂。这个项目旨在创建一个轻量级、文件驱动的博客系统，让写作回归本质。

## 技术栈

- **Next.js 16** — App Router + Server Components
- **MDX** — 在 Markdown 中使用 React 组件
- **Tailwind CSS v4** — 精密的视觉设计系统
- **TypeScript** — 类型安全的内容管理

## 设计理念

博客的视觉设计采用了"精密触感"（Precision Tactility）设计语言——工业精密美学与触觉反馈的结合。每个 UI 元素都经过精心调校，从按钮的 3D 压缩效果到玻璃态导航栏。
